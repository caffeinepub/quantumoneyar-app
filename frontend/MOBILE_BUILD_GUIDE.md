# Quantumoney AR - Native Mobile Build Guide

This guide provides step-by-step instructions for generating native Android APK and iOS packages for the Quantumoney AR application.

## Prerequisites

### Required Software
- Node.js 18+ and npm/pnpm
- Android Studio (for Android builds)
- Xcode 14+ (for iOS builds, macOS only)
- Capacitor CLI: `npm install -g @capacitor/cli`
